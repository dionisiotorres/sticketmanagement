Generate serial/lot number based from MO

Generate serial/lot number based from MO is useful to generate custom serial no for manufacturing order by user.

Features

User can set digits

You can set number of digits for serial no.

Helps to make serial/lot number

You don't have to create serial/lot number manually.

User can set prefix

You can set number prefix as you wish for serial no.

Users don't need to create serial/lot number

It will create new serial/lot number for the product in manufacturing order as per your input.

MO Serial Number Configuration

Under Manufacturing > Configuration > Settings menu you can see serial no configurations.

You have options to set MO Serial Number.

1. Global Option :

If you select "Global" option then in MO, Product lot/serial number is set based on global digit and prefix.

2. Product Wise Option :

If you select "Product Wise" option then in MO, Product lot/serial number is set based on particular product digit and prefix.

Enable "Lots & Serial Numbers"

Make sure that in settings of Inventory "Lots & Serial Numbers" is enable


Tracking as "By Unique Serial Number"

In product select Tracking as "By Unique Serial Number" And create Bill of Materials for the product.


Create Bill of Materials


Create Manufacturing Order

Now you have to click produce button for same number of times as Manufacturing Order product quantity.


Produce Wizard

Click on Record Production button to produce one product with custom serial/lot number.


You can see the finished products with global serial/lot no.


Product Wise MO Serial Number

Under Manufacturing > Configuration > Settings menu you can see serial no, Select Product Wise option.


Set Product Digit and Prefix

Set product digit and prefix under inventory tab.


Create Manufacturing Order

Now you have to click produce button for same number of times as Manufacturing Order product quantity.


You can see the finished products with product wise serial/lot no.
